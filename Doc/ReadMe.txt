EmailSender Feature Guide (Draft)

1. Vertical Layout Import File
To import email address from a vertical layout file, go to main menu Recipients->Import External File...
On the Import Recipients dialog box, set the Files of type to "Vertical text files(*.txt).
Browse to the file and click the Open button.

The format of the file must be plain text. Contacts are seperated by at least one blank line. Each item occupies one line. Line ends by crlf (\r\n). 
Line are not start from the list of labels are ignored furing import.
Labels are case-sensitive.
"Address:" The physical address of the contact.
"Phone:" The phone number of the contact
"Fax:" The fax number of the contact
"E-mail:" The email address
"Website:" Company URL
"Contact:" The contact person's name
